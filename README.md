# AI Code Mentor

基于 CLAUDE.md 和 Skills 的 AI 辅助编程教学框架。

## 目标

根据用户水平自适应选择教学模式，既能帮中级程序员戒掉 AI 依赖，也能帮新手通过观察好代码建立心智模型。

## 两种教学模式

| | 引导模式 | 示范模式 |
|---|---------|---------|
| **适合** | 有基础的中级程序员 | 接触新技术栈/新领域 |
| **谁写代码** | 用户写，导师引导审查 | 导师写，用户学习理解 |
| **教学方式** | 追问 → 拆步骤 → 审查 | TDD 循环 → 讲解 → 举一反三 |
| **目标** | 锻炼独立思考，防止能力退化 | 建立好代码的心智模型 |

启动 `/mentor` 时会询问用户水平，自动选择模式。随时可说「切换模式」。

## 使用方式

将 `CLAUDE.md` 和 `.claude/commands/` 目录放入你的工作项目中，然后使用：

- **`/mentor`** — 描述一个功能需求，导师根据你的水平选择引导或示范模式
- **`/review`** — 指定文件或描述 bug，导师引导你发现和修复问题

### 用户指令词

| 指令 | 含义 |
|------|------|
| `开启教学` | 打开自动教学检测，无需手动输入命令 |
| `关闭教学` | 关闭自动教学检测，回到手动模式 |
| `继续` | 进入下一个任务 |
| `切换模式` | 引导 ↔ 示范 |
| `我来写` | 角色反转：导师写测试，你写实现 |
| `复盘` | 阶段性总结：模式、坑、重构建议 |

有问题直接问即可，不需要额外的指令词。
